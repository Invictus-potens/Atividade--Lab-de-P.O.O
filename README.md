### Laboratório de Programação Orientada a Objetos
## Projeto Prático - Gestão de Campeonato de Futebol
Este projeto consiste em um software desenvolvido como avaliação parcial para a disciplina de Laboratório de Programação Orientada a Objetos na Unaerp (Quinta Etapa - Engenharia de Software).

## Objetivo
Implementar um software que gerencie pontos para um grupo de usuários em relação às partidas de um Campeonato de Futebol.
- Aplicar os conceitos fundamentais do paradigma de programação orientado a objetos: encapsulamento, construtores (padrão e sobrecarregado), herança simples, polimorfismo (sobrecarga e sobreposição).
- Específicos (LPOO): aplicar melhores práticas de desenvolvimento, implementação adequada dos pilares de O.O., interfaces de usuário.
